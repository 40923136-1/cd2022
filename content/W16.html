W16

控制機械手臂的步驟:

1.開啟啟動檔，並在Scite(小白板)開啟uarm_tkinter_control.py這個檔案

2.開啟之後，在小白板上方有一個Tools的選項，點進去之後有一個Go的選項，按下去就會跑出一個控制面板。

3.在coppeliasim.exe打開uarm_nx_tkinter_gul_control_exhuastive.simscene.xml

4.在控制面板按下To connect後，就可以控制機械手臂

利用 OpenCV 擷取場景中的 Vision Sensor 串流影像:

w15_rotating_cuboid.7z

  1. 解開壓縮檔案, 配合 rotating_cuboid.py 中的第 17 行, 修改 coppeliaSim.exe 所在目錄, 程式設定位於 C:/CoppeliaSimEdu_4.3.0_rev3 目錄中.
  2. 確定 coppeliaSim.exe 位置設定正確後, 以 SciTE 執行 rotating_cuboid.py, 將會以 headless 模式 (無 GUI 介面) 啟動 rotating_cuboid.ttt, 並將其中的兩個 Vision Sensor 串流影像取出播放.
  3. 操作者按下 p 鍵, 將會暫時 pause 場景模擬.
  4. 操作者按下 s 鍵, 將會重新啟動場景模擬.
  5. 操作者按下 e 鍵, 將會關閉 CoppeliaSim 與 Python 並退出模擬系統.
  6. rotating_cuboid.ttt 中控制 Cuboid 旋轉的程式採 Python scripting 格式編寫.
